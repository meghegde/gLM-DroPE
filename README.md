# gLM-DroPE



This repository applies the DroPE (https://github.com/SakanaAI/DroPE/tree/main) methodology to BERT-based genomic language models (gLMs) for non-coding variant effect prediction from human DNA sequences.

Many thanks to the authors of the original study for making their code and methodology available. Their paper can be found here: https://doi.org/10.48550/arXiv.2512.12167.
